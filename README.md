# FASTAPI Fundamentals
Demo code for the course "FastAPI Fundamentals" on Pluralsight.


## Installing

### Python 3.10
pyenv install 3.10.6
pyenv versions
python --version
pyenv global 3.10.6  
python --version


### FastAPI
python3 -m venv venv
pip3 install fastapi[all]  


## Running
uvicorn carsharing:app --reload
